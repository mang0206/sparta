# week-01-project

#### **1. 제출 방법**

##### 1.1. **Branch 생성 및 작업**
제공된 주차별 Git repository에서 **신규 Branch**를 생성한 뒤 작업합니다.

- **Branch 이름 형식**  
  `“work/{휴대전화번호}-{영문 이름}”`  
  **예시:** `work/1234-ji-hye`

##### 1.2. **Commit 및 Push**
작업 내용을 **작업용 브랜치**에 Commit한 뒤 Push합니다.

##### 1.3. **PR 요청**
작업이 완료되면 **작업용 브랜치**에서 **제출용 브랜치**로 PR(Pull Request)을 생성합니다.

- **제출용 브랜치 이름 형식**  
  `“project/{휴대전화번호}-{영문 이름}”`  
  **예시:** `project/1234-5678-ji-hye`

##### 1.4. **PR 병합**
프로젝트 이상이 없다면 **제출용 브랜치**에 PR을 병합합니다.
